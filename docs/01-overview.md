# 1. Обзор приложения

## Назначение

**Relationship Reminder Bot** — приватный Telegram-бот для пары (парень и девушка), который помогает:

- **Организатору (OWNER)** — не забывать важные даты (годовщины, дни рождения и т.д.) и видеть пожелания партнёра
- **Партнёру (PARTNER)** — записывать пожелания, идеи, мечты, чтобы вторая половинка могла их прочитать

## Основные возможности

| Роль | Возможности |
|------|-------------|
| **OWNER** | Добавление дат с напоминаниями, просмотр дат, просмотр пожеланий партнёра, получение напоминаний |
| **PARTNER** | Добавление пожеланий/заметок, просмотр своих заметок |

## Сценарии использования

1. **Организатор** добавляет годовщину с напоминанием за 10 дней — бот напомнит в нужную дату
2. **Партнёр** пишет «Хочу сходить в новый ресторан» — организатор видит это в разделе «От второй половинки»
3. **Случайные напоминания** — бот периодически напоминает организатору случайное пожелание партнёра

## Целевая аудитория

Пары в отношениях: один человек — организатор (помнит даты, получает напоминания), второй — партнёр (записывает желания и идеи).

## Специфика

- **Приватный бот** — доступ только для whitelist (OWNER_ID, PARTNER_ID)
- **Режим разработки** — `DISABLE_WHITELIST=true` отключает whitelist, любой пользователь может выбрать роль
- **Тон коммуникации** — дружеский, в контексте отношений («твоя половинка», «вас двоих»)
